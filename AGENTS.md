# Ent

Immutable agent ops kit. Clone into `ent/` inside a workspace; open the **workspace root** in Cursor.

| Need | Path |
|------|------|
| Build plan | `PLAN.md` |
| Boot | `ENT-BOOT.md` |
| Execution protocol | `docs/EXECUTION.md` |
| Testing gates | `docs/TESTING.md` |
| Manifest contract | `ent.manifest.yaml` |
| CLI | `node tools/ent.mjs --help` |
| Ent dev workspace file | `ent-dev.code-workspace` (sibling `Ent-workspace-test/`) |

Onboard: **`/ent-onboard`** (from a **consumer** workspace root, not this repo alone).
